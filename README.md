SPinSynth-T

Monophonic software synthesizer developed by Ricardo Peculis.

SPinSynth-T is not a finished product. It was created as an experimental software synthesizer project using the Teensy 3.1 microcontroller platform.

To compile this scketch, select the Teensy board as 3.2/3.1 and USB Type as Serial+MIDI.

The project was developed to explore real-time digital sound synthesis on embedded hardware. The oscillator, filter, amplifier, and envelope generator were implemented from scratch without using the Teensy Audio Library developed by Paul Stoffregen of PJRC.

SPinSynth-T does, however, make use of the PJRC MIDI and USB-MIDI libraries.

The /docs folder contains the SPinSynth-T circuit diagram, MIDI control documentation, and several images of the assembled hardware. Additional documentation describing the architecture and functionality of the synthesizer will be added in the future.
